What I Do

Compositing

Live action compositing and CG intergration,Keying, rotoscoping, BG prep and paint out, set extension, 2D & 3D tracking, 3D projections, colour correction, stereo prep & correction using Ocula, Python scripting and gizmo development.

FX

I am fluent in simulation and procedural workflows using Houdini. I have experience setting up tools and digital assets to optimize productions. Creativity, problem solving, renderfarms, distributed simulations, and python are used often to solve complex problems and deliver art directed results.

Web Design

I like to code things from scratch, and enjoy bringing ideas to life in the browser; Languages: HTML,CSS,JavaScript,Bootstrap 4; Dev Tools: Atom,GitHub,Terminal

clean code

An essential tool for a technical artist. Python, Bsh, HTML, CSS, Gdscript, and Vex are languages I am using most often to streamline production and create innovative toolsets.

Photoshop

2D/3D matte painting for films and television. Photoshop has become the the industry standard for graphics designers, digital artists and web designers.
